Avatar

An avatar is a graphical representation of a user or the user's character.

Interactive demonstration

Usage

md-accordion has a [mdAccordionHeadline] attribute that you can use to set the headline and the rest of the content is the body.

API

Inputs

Property Type Default Description
src string undefined The Avatar's image source.
fullName string undefined The Avatar's full name to get the initial from. It is also used as an alt if src is specified.
size number undefined The Avatar's size.
href string undefined The Avatar's URL.
target string undefined The Avatar's URL target.
interactive boolean false Whether or not the Avatar is interactive.
