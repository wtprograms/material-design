Accordion

An accordion is a vertically stacked list of items that utilizes show/hide functionality to reveal content.

Usage

md-accordion has a [mdAccordionHeadline] attribute that you can use to set the headline and the rest of the content is the body.

API

Inputs

The Accordion does not have any inputs.

Properties

Property Type Default Description
openCloseState OpenCloseState 'closed' Whether the Accordion is open or closed.
