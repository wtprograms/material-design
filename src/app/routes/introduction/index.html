Introduction

Material Design components is a library of components that follows Google's Material Design 3 guidelines. This website received some of its contents from material-web.dev.

What is Material Design?

Material Design is a design system built and supported by Google designers and developers.

The latest version, Material 3, enables personal, adaptive, and expressive experiences â€“ from dynamic color and enhanced accessibility, to foundations for large screen layouts and design tokens.

For whom are these components?

These components are for Angular websites that want to follow Google's Material Design 3 guidelines.

Many of the components in this library are drop-in replacements for browser elements like button and input.

What are tokens?

Design tokens are the building blocks of a design system. They are custom CSS properties that store design values like colors, typography, spacing, etc.
